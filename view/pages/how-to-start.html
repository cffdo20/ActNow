Como começar?

Criar um Projeto Social pode ser uma tarefa desafiadora, por isso, o tutorial abaixo visa explicar em detalhes sobre o porquê criar um Projeto Social e como usar o ActNow para gerenciá-lo após a criação. Os seguintes tópicos serão abordados:

  1. O que são Projetos Sociais
  2. Como escolher uma causa
  3. Como fazer o planejamento
  4. Como administrar o Projeto

O que são Projetos Sociais?

Projetos sociais são iniciativas criadas para ajudar a melhorar a vida de pessoas e comunidades. Eles têm como objetivo resolver problemas sociais e promover mudanças positivas.
